# Métodos númericos

Aplicación que permite hallar las raíces de funciones matemáticas utilizando diversos métodos numéricos.

## Tecnologíascrequeridas

- **Lenguaje principal**
  - python `>=3.12.1`

- **Instalador de paquetes**
  - pip `>=23.2.1`

## Instalación

- **Clonar el repositorio:**

  ```bash
  https://github.com/BrayanZipa/metodos_numericos.git
  cd metodos_numericos
  ```

- **Crear entorno virtual en Python:**

  ```bash
  python -m venv .venv
  ```

- **Activar entorno virtual:**

  ```bash
  .venv\Scripts\activate
  ```

- **Instalar las dependencias requeridas:**

  ```bash
  python -m pip install -r requirements.txt
  ```

- **Ejecutar programa:**

  ```bash
  python src/main.py
  ```
